# 시계 내부 구조 검토 — 명품 무브먼트 기준

## 현재 구현된 디테일

| 항목 | 구현 여부 | 설명 |
|------|-----------|------|
| **메인 플레이트** | ✅ | 기반판(plate), 기어들이 올라가는 토대. 현재 단일 원형 플레이트. |
| **브릿지** | ✅ | 톱니바퀴·밸런스를 고정하는 상부 부품. 앙글라주(모서리 비스듬히 깎기) 형태로 하나 구현. |
| **쥬얼 (루비 베어링)** | ✅ | 피벗 지점의 마찰 감소용 보석. 빨간 구형으로 5개 배치. |
| **기어 트레인** | ✅ | 중심 톱니(분침), 제3·제4(초침), 이스케이프 휠까지 톱니 형태로 표현. |
| **밸런스 휠** | ✅ | 링 형태로 구현, 좌우 진동(틱톡) 애니메이션. |
| **헤어스프링** | ✅ | 밸런스와 함께 움직이는 나선형 스프링. 3D 튜브로 나선 표현. |
| **이스케이프 휠** | ✅ | 이스케이프먼트의 한 부분으로 회전 애니메이션. |

## 명품 무브먼트에서 흔히 쓰이는 추가 디테일 (향후 보강 시 참고)

- **팔레트 포크 (Pallet fork)**  
  이스케이프 휠과 밸런스 사이에서 한 틱씩 잠그는 Y자형 부품. 시각적으로 넣으면 “한 틱마다 톱니가 걸리는” 느낌을 줄 수 있음.

- **배럴 (Barrel)**  
  메인스프링이 들어 있는 통. 오토매틱이면 로터(무게추)와 연결. 스켈레톤에서는 보통 12시 방향이나 한쪽에 큰 원형으로 보임.

- **브릿지 개수·형태**  
  실제 고급 무브먼트는 “배럴 브릿지”, “센터 휠 브릿지”, “밸런스 콕(Coq)” 등 부품별로 브릿지가 나뉘고, 각 브릿지에 **블루 스크류**나 **골드 스크류**가 보이는 경우가 많음.

- **마감**  
  - **제네바 스트라이프 (Côtes de Genève)**  
    플레이트·브릿지 표면의 줄무늬 마감.  
  - **펄라주 (Perlage)**  
    작은 원형 연마 자국.  
  - **앙글라주**  
    브릿지 가장자리 45° 비스듬히 깎기. (현재 브릿지 형태로 일부 반영됨.)

- **쥬얼 수**  
  일반적으로 17~25쥬얼. 피벗마다 쥬얼을 두면 개수가 늘어남.

- **로터 (오토매틱)**  
  오토매틱이면 뒤케이스 쪽에 반원형 무게추(로터)가 있고, 브릿지에 “AUTOMATIC” 또는 칼리버명이 각인되는 경우가 많음.

## 요약

- **겉모습**: 파일럿 워치 스타일의 딥 블루 다이얼, 아라비아 숫자, 칼침형 핸즈, 크라운, 날짜창, AUTOMATIC / SWISS MADE 문구를 3D로 반영했고, 드래그로 회전 가능.
- **내부 구조**: 명품 스켈레톤에서 중요한 요소인 **플레이트, 브릿지, 쥬얼, 기어 트레인, 이스케이프, 밸런스, 헤어스프링**은 모두 포함되어 있음.
- **추가로 넣으면 더 “명품답게” 보이는 요소**: 팔레트 포크, 배럴(또는 로터), 브릿지/스크류 수 증가, 플레이트·브릿지에 제네바 스트라이프·펄라주 텍스처.

위 항목들을 단계적으로 추가하면 명품 시계들이 사용하는 디테일에 더 가까워질 수 있습니다.
